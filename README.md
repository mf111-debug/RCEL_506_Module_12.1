# Ecobici Mexico City App

A Streamlit web app built with Python to explore and visualize the **Ecobici bike share system in Mexico City**.  
This project uses open bike-share data to display station locations and support simple interactive map-based analysis.

## Project Overview

This app connects to Ecobici-related bike share data and helps users explore station information through an interactive interface.  
It was developed using **Python**, **Streamlit**, and **Folium** for map visualization.

The main purpose of the app is to:
- visualize bike share stations in Mexico City
- interact with station data
- highlight selected stations on a map
- provide a simple example of urban mobility data analysis

## Features

- Interactive map of Ecobici stations
- Station markers displayed using Folium
- Highlight a selected station
- Streamlit interface for easy exploration
- Built with Python and open data sources

## Technologies Used

- **Python**
- **Streamlit**
- **Pandas**
- **Folium**
- **ipywidgets** *(if used during notebook testing/prototyping)*

## Data Source

This project uses open bike share data based on the **General Bikeshare Feed Specification (GBFS)**.
